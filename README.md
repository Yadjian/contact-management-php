# contact-management-php

Application de gestion de contacts (PHP).

## Stack Technique
* Langage : PHP
* Architecture : MVC (Model, View, Controller)
* Base de données : MySQL

$server = " ... ";

* php -S localhost:8000

* http://localhost:8000
